1. blocked on a simple problem:

What was the problem:
I was stuck on a problem called superfizzbuzz. Basically I had to write a function called super_fizzbuzz which takes an array of numbers as its input. It then loops over each number in the list, checking to see if it's divisible by 3, by 5, or by both. Then based on this check, it will add "Fizz", "Buzz", "FizzBuzz", to a new list. When it's finished looping over all the numbers, it returns the new "fizzbuzzed" list. The program does that for you and gives you back a new list with all the changed numbers.

What problem-solving techniques did you use:
Even though it wasn’t a large challenge, it was enough for me to get overwhelmed. Therefore, when I realized this was going to be a bit more challenging for me, I decided to break it down into small sections for me to understand. I read over the instructions step by step, breaking it down. I then did a lot of research on the web to find something similar. When I couldn’t find something, I reached out for help, and even watched a zoom livestream to ask what to do because I was lost. I continued more searching on youtube, and google, and managed to figure it out. So a lot of persistence.

How did you feel throughout the process:
It definitely involved a lot of down moments, when I was like well crap this looks impossible. But with the help of some food and exercise breaks, I was able to look at it again, and knowing that we could actually use .map, helped a lot, and I managed to finish it…after 5 hours, but hey it’s done now yay!

What did you learn:
I learnt that even though it may take me a very long time to get something, or I still don’t fully understand it, there will be those light bulb moments when you’re like damn wow I actually get it!

2. elegantly solved a problem:

What was the problem:
It was the ‘capitalize’ section in the Kata. Imagine you have a sentence, and you want to make it look bouje by capitalizing the first letter of each word. This function takes that sentence as a string and uses something called .map() to go through each word in the sentence. Then for each of the words, it capitalizes the first letter. And then, it returns the modified sentence as a string, which means the output is still a sentence, but now with each word starting with a capital letter.

What problem-solving techniques did you use:
I broke it down into small sections. Researched the question to try and find something similar, created a step-by-step plan for how I would carry it out. And wrote the actual code.

How did you feel throughout the process:
I really felt like I had a light bulb moment, I finally figured out where the const, slice, split all come in. It made me feel excited for a few seconds.

What did you learn:
I learnt that although I had such a hard time the day before, continuing to learn is far better than anything else. Step by step. Don’t worry about how fast others are going, focus on my own pace.
